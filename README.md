# rhweb-desktop

> Instalação dos componentes utilizados:
##### *npm install express*
##### *npm install sequelize*
##### *npm install -d sequelize-cli*
##### *npm install mysql2*
> *Inicialização*:
##### *npm -y init*
##### *npx sequelize init*

> Executando os migrations (sequelize ORM):
##### *sequelize db:migrate*
